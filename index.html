Random Forest Index Futures Forecaster

Built a prototype classification model in Python using Bloomberg API data to forecast directional movement (up/down/flat) of Southeast Asian index futures based on features like volatility and volume. Initial tests showed ~80% directional accuracy on historical data, though likely subject to overfitting. Created during a summer internship on the equities desk at a bulge bracket investment bank. After I returned to university, the model was further refined and used by the trader desk-head in live decision-making.

Technologies: Python, VBA
